# 花园种植游戏 MVP - Agent工作框架

## 项目概述

- **项目名称**: 花园种植游戏
- **类型**: 2D休闲益智网页小游戏
- **核心玩法**: 播种 → 浇水 → 开花 → 收获
- **技术栈**: Vite + React + TypeScript

---

## Agent架构

### 总控Agent (Orchestrator)

**职责**:
1. 接收用户需求
2. 分配任务给各职能Agent
3. 审核各Agent产出
4. 记录进度
5. 确保质量

**审核标准**:
- 产出是否符合需求
- 代码/文档是否规范
- 是否有遗漏

---

### 职能Agent

| Agent | 职责 | 输入 | 输出 |
|-------|------|------|------|
| 策划Agent | 游戏设计、规则、数值 | 用户需求 | design/*.md |
| 美术Agent | 素材产出 | 策划文档 | art/*.* |
| 编程Agent | 代码实现 | 策划+美术 | build/*.* |

---

## 工作流程

```
[用户需求]
     ↓
[总控Agent] ←→ [策划Agent] → 产出设计文档
     ↓
  审核 ↓
     ↓
[总控Agent] ←→ [美术Agent] → 产出素材
     ↓
  审核 ↓
     ↓
[总控Agent] ←→ [编程Agent] → 产出代码
     ↓
  审核 ↓
     ↓
   [完成]
```

## 游戏操作说明

1. **播种**: 点击工具栏花朵 → 拖拽到空花盆
2. **浇水**: 点击浇水 → 拖拽到已播种花盆
3. **收获**: 点击开花花朵或拖拽收获工具
4. **仓库**: 右上角显示收获数量
