### P2.5 — 数值循环扩展（长线策划方向）

> **核心理念**：增加更多「产出 ↔ 消耗」闭环，让玩家始终有目标、有成长、有收获。

#### 新增产出渠道
- [ ] **季节活动花朵**：每月/每季度限定花朵，过期后无法再种，激励持续登录
- [ ] **花卉合成 / 杂交**：消耗 2~3 种已有花朵合成稀有新品种，产出更高价值花朵
- [ ] **花束组合系统**：将多朵花组合成花束出售，组合加成倍率（1.5x~3x），鼓励多样化种植

#### 新增消耗渠道
- [ ] **花园扩建**：花盆从 28 个逐步扩展（需大量金币解锁新区域：4→8→12→16→20→24→28→32）
- [ ] **肥料 / 加速道具**：消耗金币购买，缩短冷却/生长时间，提升产量
- [ ] **花盆强化**：花盆升级消耗金币，提升浇水效率/冷却速率/连续收割次数

#### 数值深度扩展
- [ ] **花朵稀有度体系**：普通→优良→稀有→史诗→传说，同种花不同品质影响售价倍率
- [ ] **连续收获加成**：同一花盆连续收获同种花给予递增奖励（连击 combo），断种重置
- [ ] **每日/每周挑战任务**：完成指定种植/收获目标获得额外奖励，增加短期目标
- [ ] **花卉市场波动**：不同花朵售价随时间浮动（±20%），鼓励玩家观察最佳出售时机
- [ ] **花园声望系统**：累积声望解锁高级功能（自动浇水、批量升级），作为长期追求目标
- [ ] **图鉴完成度奖励**：按百分比里程碑（25%/50%/75%/100%）发放永久加成 buff
