# 仓库界面设计方案

## 1. 界面结构

### 工具栏新增仓库按钮
- 位置：工具栏右侧
- 图标：📦
- 点击弹出仓库面板

### 仓库面板
- 弹出方式：从下方滑出
- 样式：网格背包
- 关闭方式：点击遮罩/再次点击按钮

---

## 2. 仓库网格设计

### 布局
- 网格：4列 × 4行 = 16格
- 每格：正方形自适应
- 滑动：支持竖向滑动

### 格子内容
- 有物品：显示花朵图标 + 数量
- 无物品：显示空置/占位符

### 统计信息
- 顶部显示：总计收获数量
- 底部显示：花朵类型统计

---

## 3. 交互

| 操作 | 效果 |
|------|------|
| 点击仓库按钮 | 弹出仓库面板 |
| 点击遮罩 | 关闭仓库面板 |
| 滑动仓库 | 浏览更多格子 |

---

## 4. 视觉

- 面板背景：毛玻璃效果
- 格子：圆角方格
- 动画：滑入滑出
