# 移动端适配与特效优化设计方案

## 1. 移动端适配

### 1.1 视口设置
- 禁止缩放: `user-scalable=no`
- 理想视口: `width=device-width, initial-scale=1`

### 1.2 布局调整

| 元素 | 桌面端 | 移动端 |
|------|--------|--------|
| 花盆网格 | 4×7固定 | 自适应缩放 |
| 工具栏 | 横向排列 | 纵向紧凑 |
| 花盆大小 | 75×85px | 响应式计算 |
| 间距 | 6px | 自适应 |

### 1.3 触摸优化
- 增大触摸区域
- 移除hover效果（移动端无hover）
- 支持触摸滑动

---

## 2. 特效设计

### 2.1 播种特效
- 种子落下动画（translateY + bounce）
- 粒子飞溅效果

### 2.2 浇水特效
- 水滴下落动画
- 花朵湿润闪烁

### 2.3 开花特效
- 花朵绽放缩放动画
- 光芒散射效果
- 星星/爱心飞出

### 2.4 收获特效
- 花朵上升消失
- +1飘字动画
- 金币/光芒效果

### 2.5 UI特效
- 工具选中发光
- 按钮点击反馈
- 仓库数量增加动画

---

## 3. 响应式断点

| 断点 | 宽度 | 调整 |
|------|------|------|
| mobile | < 480px | 单列工具栏，小花盆 |
| tablet | 480-768px | 双列工具栏 |
| desktop | > 768px | 默认布局 |

---

## 4. 性能优化

- CSS动画替代JS动画
- 使用transform和opacity
- 减少重绘重排
- 移动端降低特效
