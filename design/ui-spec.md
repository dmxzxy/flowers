# UI规格文档 - 花园种植游戏 MVP

## 1. 屏幕布局

### 1.1 主场景

```
┌─────────────────────────────────┐
│         背景：花棚+天空          │
├─────────────────────────────────┤
│  [花盆1][花盆2][花盆3][花盆4]    │  ← 可见4列
│  [花盆5][花盆6][花盆7][花盆8]    │
│  [花盆9]...                     │  ← 可横向滑动
│  ...                            │
│                                 │
│  [◀ ◀ ◀] 或 滑动指示器          │  ← 滑动提示
└─────────────────────────────────┘
```

### 1.2 种子选择面板

弹出位置：屏幕底部或中央

```
┌─────────────────────────────────┐
│         请选择种子              │
├─────────────────────────────────┤
│  [玫瑰] [郁金香] [雏菊] [向日葵] │
│                                 │
│      [取消]                     │
└─────────────────────────────────┘
```

## 2. 组件规格

### 2.1 花盆组件 (Pot)

- **尺寸**：200×200 像素（显示区域）
-整个 **点击区域**：花盆
- **状态样式**：
  - empty：空花盆，显示花盆图片
  - seeded：显示种子图片
  - growing：显示嫩芽图片
  - blooming：显示开花图片

### 2.2 花盆网格 (PotGrid)

- **布局**：4列 × 7行
- **滚动**：水平横向滚动
- **容器宽度**：4列 = 800px（200×4）
- **滚动方式**：触摸滑动 / 鼠标拖动

### 2.3 种子选择面板 (SeedPicker)

- **触发**：点击空花盆
- **显示**：4个种子选项
- **交互**：点击选择，确认播种
- **关闭**：选择后自动关闭 或 点击取消

### 2.4 浇水按钮

- **触发**：点击已播种的花盆
- **效果**：显示浇水动画 → 花开
- **位置**：花盆上方或点击位置

## 3. 交互流程

### 3.1 播种流程

```
1. 用户点击空花盆
2. 弹出种子选择面板
3. 用户选择种子类型
4. 面板关闭，花盆显示种子状态
5. 数据更新：pot.state = 'seeded'
```

### 3.2 浇水流程

```
1. 用户点击已播种的花盆
2. 显示浇水动画（可选）
3. 花朵状态更新为 blooming
4. 数据更新：pot.state = 'blooming'
```

## 4. 响应式设计

- **移动端优先**：触摸操作
- **桌面端**：鼠标点击 + 滚动条
- **横屏推荐**：最佳显示4列

## 5. 动画效果

| 场景 | 动画 |
|------|------|
| 种子落下 | 轻微弹跳 |
| 浇水 | 水滴下落 |
| 花开 | 放大绽放 |
| 滑动 | 平滑滚动 |
