# 游戏设计文档 - 花园种植游戏 MVP

## 1. 游戏概述

一款2D休闲益智类网页小游戏，玩家在花棚中种植花朵，体验从播种到收获的乐趣。

## 2. 核心玩法

### 2.1 玩法流程

```
[选择花盆] → [选择种子] → [播种] → [浇水] → [鲜花绽放]
```

### 2.2 交互说明

1. **选择花盆**：点击任意空花盆
2. **选择种子**：弹出种子选择面板，选择要种植的花种
3. **播种**：种子落入花盆，显示种子状态
4. **浇水**：点击已播种的花盆，触发浇水操作
5. **绽放**：浇水后花朵立即绽放

## 3. 场景设定

### 3.1 背景

- 花棚内部场景
- 可见部分天空（晴朗白天，蓝色渐变）
- 花棚框架、架子等元素

### 3.2 花盆布局

- **数量**：4列 × 7行 = 28个花盆
- **排列**：水平滚动查看全部花盆
- **初始状态**：全部为空花盆

## 4. 种子类型

| 种子名称 | 特点 | 颜色 |
|----------|------|------|
| 玫瑰 Rose | 经典爱情之花 | 红色 |
| 郁金香 Tulip | 优雅高贵 | 黄色/粉色 |
| 雏菊 Daisy | 清新可爱 | 白色/黄色 |
| 向日葵 Sunflower | 阳光积极 | 金黄色 |

## 5. 花朵成长阶段

每个花朵有4个状态：

| 状态 | 名称 | 说明 |
|------|------|------|
| seed | 种子 | 刚播种，显示土堆或小种子 |
| sprout | 嫩芽 | 冒出土的绿色小芽 |
| growing | 成长中 | 长出茎叶，逐渐长高 |
| bloom | 开花 | 鲜花绽放，完成生长 |

## 6. 数值设计

| 参数 | 值 |
|------|-----|
| 花盆数量 | 28 (4×7) |
| 种子类型 | 4种 |
| 浇水到绽放 | 即时（MVP简化） |
| 滑动方向 | 水平横向 |
